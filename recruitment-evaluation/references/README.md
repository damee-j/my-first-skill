# 참조 문서 배치 안내

채용 평가 스킬이 사용할 참조 문서를 아래 폴더에 배치해 주세요.

## 폴더별 배치 안내

### `jd/` — 직무기술서 (Job Description)

각 직무의 JD 파일을 넣어주세요. 파일명에 직무명을 포함하면 자동 매칭됩니다.

```
jd/
├── PM-JD.pdf                    # Product Manager JD
└── Product-Designer-JD.docx     # Product Designer JD
```

**포함되어야 할 내용**: 직무 개요, 주요 업무, 필수 자격요건, 우대사항

### `criteria/` — 역량 평가기준

직무별 내부 역량 평가기준 문서를 넣어주세요.

```
criteria/
├── PM-역량평가기준.pdf
└── Designer-역량평가기준.pdf
```

**포함되어야 할 내용**: 평가 항목, 항목별 정의, 등급/점수 기준

### `interview-forms/` — 면접평가표 양식

면접 시 사용하는 내부 평가표 양식을 넣어주세요.

```
interview-forms/
├── 1차면접-평가표.pdf
└── 1차면접-평가표.docx
```

**포함되어야 할 내용**: 평가 항목, 점수 척도, 합격 기준(컷오프 점수)

## 지원 파일 형식

- PDF (`.pdf`)
- Microsoft Word (`.docx`, `.doc`)

## 후보자 자료

후보자의 이력서와 포트폴리오는 `candidates/` 폴더에 넣어주세요.

```
candidates/
├── 홍길동-이력서.pdf
├── 홍길동-포트폴리오.pdf
├── 김철수-이력서.pdf
└── 김철수-포트폴리오.pdf
```
