# 클로바노트 전문 형식

## 구조

```
{회의 제목}
{YYYY.MM.DD 요일 오전/오후 H:MM ・ 녹음시간}
{참석자1} {참석자2} {참석자3} ...

{발언자} {MM:SS}
{발언 내용 (여러 줄 가능)}

{발언자} {MM:SS}
{발언 내용}
...

clovanote.worksmobile.com
```

## 파싱 규칙

- 1행: 회의 제목
- 2행: 날짜/시간 메타데이터. `・` 구분자 기준 왼쪽=시작시각, 오른쪽=총 녹음시간
- 3행: 참석자 목록 (공백 구분)
- 본문: `{이름} {MM:SS}` 패턴으로 발언 블록 시작. 다음 발언 블록 전까지가 해당 발언 내용
- 마지막 행: `clovanote.worksmobile.com` (무시)
- "참석자 N" 형태의 이름은 미식별 참석자 (음성 인식 한계)
